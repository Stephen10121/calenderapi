# API CALLS

## Authentication

/login
/signup
/validate

## Note

All calls to the backend require an Authorization header with this type of style:

```
Authorization: Bearer <Bearer Token>
```

## Group Part

`/createGroup` Creates a group.
`/joinGroup`
`/groupIdTaken`
`/groupInfo`

## Job Part

`/addJob`
`/getJobs`
